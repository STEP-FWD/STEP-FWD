# STEP Protocol Database

**Authors:** Project developed at ICFO by Ediz Herkert, Jaime Díez Mérida, Bianca Turini and Rosella Zaffino

**Created:** 26-03-2026

## Description

The objective is to have a database in which we have all the fabrication protocols made at ICFO. We have base protocols, which are peer-reviewed before being accepted and derived protocols, which add details to the base protocols.

The protocols are organized by category: 2D materials, EBL, etc.

Protocols are version-controlled, modular, and inheritable.
